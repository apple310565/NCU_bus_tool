# NCU_bus_tool
給中央大學在校生的交通APP

## 服務內容
包含即時公車、校內Youbike、火車動態

## 介面展示
* 公車
* 火車
* 校內Youbike

## 資料來源
資料介接於交通部PTX平臺資料
